MINHAS ESPECIALIDADES.

HTML

HTML (Hypertext Markup Language) "HTML é a linguagem básica de marcação usada para estruturar o conteúdo da web. Ele define a estrutura e o layout dos elementos, como textos, imagens e links, permitindo que páginas da web ganhem forma."

CSS

CSS (Cascading Style Sheets) "CSS é uma linguagem de estilo usada para definir a aparência visual de páginas da web. Com ele, é possível estilizar cores, fontes e layouts, trazendo design e personalização aos elementos estruturados pelo HTML."

PYTHON

Python "Python é uma linguagem de programação versátil, conhecida por sua simplicidade e legibilidade. Usada tanto para desenvolvimento web quanto para análise de dados, automação e inteligência artificial, é ideal para criar soluções eficientes em diversas áreas."
